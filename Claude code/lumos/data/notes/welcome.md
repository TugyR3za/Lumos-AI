# Welcome to Lumos

This is a sample note. Anything you drop in this `data/notes/` folder
(`.md` or `.txt`) becomes searchable by the assistant.

## About this project

Lumos is a private, family-oriented AI assistant. The plan is to build it
incrementally on top of open-weight models — prioritizing memory, tools, and
retrieval over raw model size. It should eventually support English, Farsi,
and Turkish, with voice, notes access, web search, and a coding agent.

## Try this

Ask Lumos: "What is the Lumos project about?" — it should answer using this
note (via the `search_notes` tool) rather than guessing.
