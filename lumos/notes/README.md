# Lumos notes folder

Place private notes and project files here. Lumos v0.1 indexes supported text and source-code formats into local SQLite FTS5 search.

Good examples:

- Markdown and text notes
- JSON, YAML, and TOML configuration notes
- Python, JavaScript/TypeScript, C/C++, and Rust source files

The index stores extracted text in `data/lumos.db`. Delete the database and restart Lumos to rebuild it from scratch.
